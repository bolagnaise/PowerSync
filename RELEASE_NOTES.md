## What's Changed

**Add Sungrow SH10RS model selection**
PowerSync now lists the Sungrow SH10RS as a selectable SH-series hybrid inverter model. This gives single-phase SH10RS installs the correct model label instead of needing to choose a three-phase SH10RT or a smaller SH-series RS model while using the existing Sungrow SH Modbus controller path.

Update available via HACS
