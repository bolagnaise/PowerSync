## What's Changed

**Sigenergy Flow Power tariff uploads now use PowerSync-adjusted prices**
Sigenergy Cloud tariff sync now applies the same Flow Power PEA, network tariff, and export-rate adjustments that PowerSync uses for its dashboard and optimiser pricing before uploading the buy/sell schedule. This keeps the Sigenergy app tariff schedule aligned with the actual PowerSync Flow Power price instead of uploading raw NEMO/AEMO wholesale values.

Update available via HACS
