## What's Changed

**Fix GloBird pricing options**
PowerSync no longer shows an unknown error when opening Electricity pricing with GloBird selected. The GloBird ZeroHero plan selector is now available in both the initial setup flow and the options flow, so existing users can update their GloBird or ZeroHero settings without the pricing dialog crashing.
