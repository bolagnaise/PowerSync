## What's Changed

**Readable LP forecast summary values**
The Home Assistant dashboard now renders the LP forecast summary with a dedicated responsive card instead of four cramped default entity cards. Solar, load, import price, and export price forecast values keep their labels and units readable at normal dashboard widths, and price averages are shown in the same minor-unit format used by the rest of the PowerSync dashboard, such as c/kWh.

Update available via HACS
