## What's Changed

**Dashboard manual battery timers are clearer**
The PowerSync dashboard now labels the shared discharge timer as applying to Discharge, Hold SoC, and Self Consumption, and the Hold SoC and Self Consumption buttons show the selected shared duration before they are pressed. This makes the recent timer consolidation visible in the UI instead of requiring users to infer it from the service wiring.

Update available via HACS
