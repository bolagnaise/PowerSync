## What's Changed

**Adaptive dashboard layout**
The Home Assistant dashboard now uses the rendered viewport width to choose one, two, or three dashboard lanes, then balances cards into the shortest lane instead of locking cards into fixed columns. This follows the same practical direction as HA layout-card/grid dashboards while keeping PowerSync's generated dashboard self-contained, reducing large blank areas on landscape tablets and desktop screens.

Update available via HACS
