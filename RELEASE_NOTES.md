## What's Changed

**Consistent config flow option text**
Initial setup and post-setup options now share the same labels and helper text for matching fields across pricing, battery, inverter, curtailment, weather, and Smart Optimization flows. This removes confusing wording differences between first install and later configuration, including Profit Max target time and target SOC.

**Unified Smart Optimization setup controls**
The initial battery setup path now shows the same Smart Optimization control set as the post-setup options page, including the optimization mode selector and 1% minimum-discharge slider resolution.
