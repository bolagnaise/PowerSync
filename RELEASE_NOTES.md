## What's Changed

**Fronius Reserva support**
PowerSync can now bridge Fronius Reserva batteries through the Fronius Modbus integration. The setup flow, options flow, coordinators, optimizer routing, BMS sensors, backup reserve controls, force charge, force discharge, idle, restore normal, and mobile-facing config/status fields are wired for the new battery system.

**Profit Max target time**
Profit Max now exposes and persists the target full-by time through setup, options, and the optimization settings API so the configured target is shown, saved, and reused consistently.
