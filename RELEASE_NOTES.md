## What's Changed

**Flow Power promo export rates**
Flow Power settings now include a dedicated Happy Hour export rate field, so users with account-specific or promotional feed-in credits can enter the real value instead of being locked to the regional default. The override is used consistently by tariff sync, current export price sensors, cost tracking, and Smart Optimization's Flow Power export window.

**Flow Power mobile app support**
The provider config API now exposes and saves the same Flow Power export-rate setting used by the mobile app, so the new mobile Flow Power export-pricing section can update Home Assistant with the same value.

Update available via HACS
